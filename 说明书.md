## 快速开始

### 启动应用

双击 `SmartRemind.app` 启动。应用会在 macOS 状态栏（屏幕右上角）显示一个图标（默认 checklist ✓）。初次启动会请求「提醒事项」访问权限，点击「允许」。

### 基础操作三步

1. **右键状态栏图标** → 点击「悬浮窗」或按 `⌥⇧R` → 桌面出现悬浮窗
2. **左键状态栏图标** → 弹出菜单面板 → 点击「设置」→ 打开主界面
3. **主界面输入框** → 输入自然语言 → 回车 → AI 解析并写入系统提醒

## 悬浮窗使用

### 基本操作

- **显示/隐藏**: 按 `⌥⇧R`
- **移动**: 鼠标拖拽悬浮窗任意位置
- **调大小**: 悬浮窗边缘拖拽（需开启「允许拖拽调大小」）
- **不会 Esc 关闭**: 悬浮窗不会因为按 Esc 而消失

### 轮播模式

悬浮窗有两种内容展示模式：

1. **翻页切换** — 每页 N 条提醒，按设定间隔自动翻页。支持 5 种切换动画：
   - 淡入淡出
   - 水平滑动
   - 垂直滑动
   - 翻转
   - 3D 旋转

2. **连续滚动** — 所有提醒连续垂直滚动，速度可调（px/s）

### 对齐系统

悬浮窗内容支持精细对齐：

- **水平对齐**: 左 / 居中 / 右
- **垂直对齐**: 上 / 居中 / 下
- **水平缩进**: 内容距离悬浮窗左边缘的像素偏移量（0-80px）
- **垂直偏移**: 内容距离顶部/底部的像素偏移量（0-60px）

### 旗标显示

带有旗标的提醒事项在悬浮窗中会以**专用旗标颜色**显示（默认橙色 `#FF9500`）。标题文字和 flag 图标均使用该颜色。可在设置→颜色中自定义。

### 插件 Widget

浮动插件显示在悬浮窗角落，可选三种内容：
- 剩余待办数（如「5项待办」）
- 当前时间（如「14:30」）
- 日期（如「6月16日 周一」）

可在设置中调整位置（四角）、字号、颜色、透明度。

## 主界面 — 提醒管理

### 布局

左侧栏：
- **提醒/设置** 分段选择器（顶部）
- 全部 / 旗标（快捷筛选）
- 按列表分组（系统提醒列表 + 数量）

右侧栏：
- AI 自然语言输入框
- 提醒列表（可编辑、完成、删除）
- 或设置面板（切换到「设置」时）

### 提醒操作

| 操作 | 方式 |
|------|------|
| 完成 | 点击左侧圆圈 |
| 编辑 | 点击铅笔图标 （或右键→编辑） |
| 删除 | 点击垃圾桶图标 （或右键→删除） |
| 新建 | 底部「新建」按钮 |
| 刷新 | 右上角刷新按钮 |
| 搜索 | 在搜索框中输入关键词 |

### AI 输入

主界面顶部的 AI 输入框支持：
- **单任务模式**: 一段文字 → 一条提醒
- **多任务模式**: 一段文字（无标点或分号分隔）→ 多条提醒

例如「明天下午3点开会 后天上午10点去超市 周五晚上聚餐」→ 多任务模式下生成 3 条提醒。

## 设置面板详解

设置面板采用**草稿系统**——任何修改不会立即生效，需要点击「应用」或 `⌘S` 保存。右侧有**实时预览窗**，所有草稿修改即时可见。

### 设置分类

#### 外观尺寸
- **宽度**: 100-600px（默认 280）
- **高度**: 36-400px（默认 72）
- **字号**: 8-28pt（默认 12）
- **副标题字号**: 6-20pt（默认 9）
- **圆角**: 0-30px（默认 12）
- **透明度**: 0.3-1.0（默认 0.85）

#### 颜色
- **背景色**: 悬浮窗背景颜色（默认深灰 `#1E1E1E`）
- **文字色**: 正文颜色（默认白 `#FFFFFF`）
- **强调色**: 轮播页码 / 按钮强调色（默认蓝 `#007AFF`）
- **旗标色**: 旗标项目专用颜色（默认橙 `#FF9500`）
- **日期色**: 截止日期文字颜色（默认橙 `#FF9500`）

所有颜色均支持 ColorPicker 点选 + Hex 值直接输入。

#### 布局对齐
- **水平对齐**: 左 / 中 / 右
- **垂直对齐**: 上 / 中 / 下
- **水平缩进**: 0-80px
- **垂直偏移**: 0-60px

#### 显示字段
- **标题行**: 旗标 / 优先级 / 标题（多选按钮）
- **副标题行**: 列表名 / 日期 / 位置 / 备注 / 标签（多选按钮）
- **角标**: 无 / 优先级 / 旗标 / 标签

#### 展示与动画
- **展示模式**: 翻页切换 / 连续滚动
- **翻页模式**:
  - 切换动画: 淡入淡出 / 水平滑动 / 垂直滑动 / 翻转 / 3D 旋转
  - 切换时长: 0.1-2.0 秒
  - 每页条数: 1-10
  - 切换间隔: 1-30 秒
- **连续滚动模式**:
  - 滚动速度: 5-200 px/s

#### 插件 Widget
- 启用/禁用
- 位置: 四角
- 内容: 剩余待办数 / 当前时间 / 日期
- 字号、颜色、透明度

#### 行为
- **悬停暂停轮播**: 鼠标移到悬浮窗上时暂停翻页/滚动
- **允许拖拽调大小**: 可用鼠标拖拽悬浮窗边缘改变尺寸
- **显示 AI 输入**: 在悬浮窗底部显示 AI 输入框
- **允许完成操作**: 在悬浮窗中直接点击完成提醒
- **默认筛选**: 全部 / 仅旗标 / 指定列表

#### 状态栏
- 图标切换: checklist / checkmark.circle / list.bullet / square.grid / bell.badge

#### 主列表
- 字号、是否显示备注/位置/日期/列表名

## AI 自然语言输入

### 支持的表达

| 类型 | 示例 |
|------|------|
| 时间 | 明天下午3点、下周三上午、今晚8点、10分钟后 |
| 日期 | 6月20日、周五、下周一、月底 |
| 地点 | 在办公室、到家时、在超市 |
| 重复 | 每周一、每天上午、每月1号 |
| 优先级 | 紧急、重要（标记为高优先级） |
| 组合 | 明天下午3点在会议室开项目评审会，每周五重复 |

### 解析流程

```
用户输入文字
    ↓
LLM API 解析 → 结构化数据 (标题/日期/位置/重复/优先级)
    ↓
Geocoding 位置 → 经纬度（如需要）
    ↓
EventKit → 写入系统提醒事项
    ↓
刷新界面
```

## AI 供应商配置

### 添加供应商

1. 主界面 → 设置 → API 供应商 → 点击「添加」
2. 填入：
   - **名称**: 方便识别的名称（如「DeepSeek V3」）
   - **API URL**: 完整的 chat completions 端点 URL（如 `https://api.deepseek.com/v1/chat/completions`）
   - **模型名称**: 对应的模型 ID（如 `deepseek-chat`）
   - **API Key**: 你的 API 密钥
3. 点击保存

### 测试连接

点击「测试连接」按钮 → 发送一条测试请求，验证 API 是否可用。

### 切换供应商

在供应商列表中点击某个供应商，当前使用的供应商旁会显示绿色勾号 ✓。

### 支持的供应商

任何 OpenAI-compatible chat completions API 均可使用，包括但不限于：
- OpenAI (GPT-4o, GPT-4o-mini)
- DeepSeek (deepseek-chat, deepseek-reasoner)
- Anthropic (通过兼容代理)
- 本地模型（Ollama, LM Studio, llama.cpp server）
- 以及其他兼容 API

## 常见问题

### Q: 悬浮窗不显示？
A: 按 `⌥⇧R`，或右键状态栏 → 悬浮窗。确保系统提醒事项权限已授予（系统设置 → 隐私与安全性 → 提醒事项）。

### Q: AI 解析失败？
A: 检查 API 供应商设置（URL、Key、模型名是否正确），点击「测试连接」。确保网络可访问 API 端点。

### Q: 修改设置后没变化？
A: 设置采用草稿系统，修改后需点击「应用」或按 `⌘S` 才会生效。右侧预览窗会实时展示草稿效果。

### Q: 如何彻底关闭悬浮窗的轮播？
A: 开启「悬停暂停轮播」，鼠标放上去即可暂停。或将展示模式设为「连续滚动」并调低速度。

### Q: 悬浮窗状态栏图标可以换吗？
A: 可以。设置 → 状态栏 → 图标，选择 checklist / checkmark.circle / list.bullet 等。

### Q: 悬浮窗可以跟随桌面切换吗？
A: 可以。悬浮窗的 `collectionBehavior` 包含 `canJoinAllSpaces`，在所有桌面都可见。
